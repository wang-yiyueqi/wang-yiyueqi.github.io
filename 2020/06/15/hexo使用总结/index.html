hexo 的整体使用方法总结

创建分页

以categories为例

  1. 创建分页： hexo new page categories
    成功后提示INFO Created: ~/Documents/blog/source/categories/index.md

  2. 打开提示路径中的index文件，并添加type属性。(layout也可以一并添加上)

  3. 给新建文章添加分页（categories）属性
    打开需要添加分类的文章， 为其添加categories属性

    注意：一篇文章只有属于一个分类，如果categories下面写了两个分类的话，则是属于分类a下的分类b中
    tags操作与之相同，
    只有添加了tags:xxx的文章才会被收录到首页的标签中。

  4. 让每次hexo new 文章名时，都能够自动生成分类或标签选项。
    打开scaffolds/post.md文件，添加你想要自动生成的选项。

创建文章

  • 新建文章 hexo new 文章名 > 缩写hexo n 文章名

其他指令

  • 新建一个网站 hexo init
  • 发表草稿 hexo publish [layout] <filename>
  • 生成静态文件hexo generate >缩写：hexo g
  • 运行服务器预览效果hexo server >缩写：hexo s
    • 重设端口 -p / –port
  • 部署到github hexo deploy >缩写：hexo d
  • 清除缓存文件 ‘hexo clean’
  • 显示Hexo版本 hexo version
